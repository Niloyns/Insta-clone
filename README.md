# InstaClone

A simple Instagram clone built with HTML and CSS.

## Description

This project is a basic clone of Instagram's interface, featuring a feed of posts and stories.

## Features

- User feed with posts
- Stories section
- Responsive design

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/InstaClone.git
   ```
2. Navigate to the project directory:
   ```
   cd InstaClone
   ```

## Usage

Open `index.html` in your web browser to view the Instagram clone.

## Technologies Used

- HTML5
- CSS3
